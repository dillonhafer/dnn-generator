# DNN Generator

The dnn_generator generates a blank DNN7 module Visual Studio project.

## How It Works

ERB Templates.

## Installation

The dnn_generator gem needs to be installed.
Ruby 1.9.2+ required.

    gem install dnn_generator

## Usage

Generate a DNN7 module project

    dnn_generator ShoppingCart
